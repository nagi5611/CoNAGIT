# CoNAGIT CLI Tool (cgit)

ディレクトリをZIP化してサーバーにアップロードするコマンドラインツールです。大容量ファイル（最大500GB）にも対応しています。

## インストール

### 方法1: インストーラーを使用（推奨）

1. `install.exe` と `cgit.exe` を同じフォルダに配置します
2. `install.exe` をダブルクリックして実行します
3. 管理者権限の要求が表示されたら「はい」を選択します
4. インストール完了後、新しいコマンドプロンプトまたはPowerShellを開きます
5. 任意のディレクトリで `cgit` コマンドが使用できるようになります

### 方法2: 手動インストール

1. `cgit.exe` を任意の場所（例: `C:\Program Files\cgit`）に配置します
2. システムの環境変数 PATH にそのディレクトリを追加します
3. 新しいコマンドプロンプトまたはPowerShellを開きます

## 使い方

### 基本的な使い方

```bash
cgit <APIキー>
```

現在のディレクトリをZIP化してアップロードします。ファイル名は自動生成されます（形式: `YYYYMMDDHHmm-<プロジェクトID>.zip`）。

### オプション

#### `-n, --name <ファイル名>`

ZIPファイル名を指定します（拡張子 `.zip` は自動で付与されます）。

```bash
cgit <APIキー> -n myproject
# → myproject.zip としてアップロードされます
```

#### `--nozip` または `-nozip`

ZIP化せずに、各ファイル・フォルダを個別にアップロードします。ディレクトリ構造がそのままサーバーに反映されます。

```bash
cgit <APIキー> --nozip
```

**注意**: `-n` オプションは `--nozip` モードでは無視されます。

## 機能

### 自動サイズ判定

ファイルサイズに応じて、最適なアップロード方式が自動選択されます：

- **20MB以下**: 通常のアップロード（`/upload-zip` エンドポイント）
- **20MB〜1GB**: Presigned URL を使用した直接アップロード
- **1GB超**: S3 マルチパートアップロード（100MB単位で分割）

### 大容量ファイル対応

最大 **500GB** までのファイルをアップロードできます。1GBを超えるファイルは自動的に分割アップロードされ、進捗が表示されます。

### 除外されるファイル

以下のファイル・フォルダは自動的に除外されます：

- `.` で始まる隠しファイル・フォルダ（例: `.git`, `.env`）
- `cgit.exe`
- `install.exe`

## APIキーの取得方法

1. [CoNAGIT](https://tech.mmh-virtual.jp) にログインします
2. プロジェクトを選択し、子プロジェクトページを開きます
3. 「APIキー取得」ボタンをクリックします
4. 表示されたJWTトークン（長い文字列）をコピーします
5. そのトークンを `cgit` コマンドの引数として使用します

**重要**: APIキーはこの時だけ表示されます。安全な場所に保存してください。

## トラブルシューティング

### 「エラー: 有効なAPIキーを入力してください」と表示される

- APIキーが正しくコピーされているか確認してください
- APIキーはJWT形式（3つの部分が `.` で区切られている）である必要があります

### 「413 Payload Too Large」エラー

- ファイルサイズが大きすぎる可能性があります
- 最新版の `cgit.exe` を使用しているか確認してください（マルチパートアップロード対応版）

### 「presigned-url取得に失敗」エラー

- サーバー側の設定（AWS S3）が正しく行われているか確認してください
- ネットワーク接続を確認してください

### インストール後も `cgit` コマンドが認識されない

- 新しいコマンドプロンプトまたはPowerShellを開いてください
- 環境変数 PATH が正しく設定されているか確認してください
- 必要に応じて、システムを再起動してください

## 技術仕様

- **対応OS**: Windows 10/11
- **最大ファイルサイズ**: 500GB
- **パーツサイズ**: 100MB（マルチパートアップロード時）
- **リトライ回数**: 最大5回（パーツアップロード時）
- **ZIP圧縮**: 無圧縮（ZIP_STORED）で高速化

## ライセンス

このツールは CoNAGIT プロジェクトの一部です。
